# 射龍門 · 撲克遊戲

網頁版單機「射龍門」（又稱 In Between／中間）。單一 HTML 檔、純前端、零依賴，雙擊即玩。

**線上遊玩**：https://nicktodj-boop.github.io/shoot-the-gate/

## 開始玩

- 直接雙擊 `index.html`，用瀏覽器開啟即可（檔案自帶所有樣式、邏輯、音效，不需網路、不需伺服器）。
- 或在資料夾內起本地伺服器：`python3 -m http.server 8777`，瀏覽器開 `http://localhost:8777`。

## 規則

開左右兩張「門柱」，下注後抽第三張，看牠能不能射進龍門。**點數大小：A=1（最小）、2–10 照數字、J=11、Q=12、K=13（最大）**，只看點數、不看花色。

| 情境 | 判定 | 賠付 |
|---|---|---|
| 第三張嚴格落在兩柱**之間** | 贏 | ×1 |
| 落在兩柱**之外** | 輸 | ×1 |
| 等於門柱（**撞柱**） | 輸 | **×2** |
| **對柱**（兩柱同點）猜大／小，猜中 | 贏 | ×1 |
| 對柱猜錯 | 輸 | ×1 |
| **三張同點** | 輸 | **×3** |
| 兩柱**相連**（如 7、8） | 自動重發、不下注 | — |
| 不想博這道門，按「**過**」 | 免費換一道新門 | — |

## 玩法機制

- **彩池（pot）**：桌上有一池彩金，每局下注額 ≤ 彩池與手上籌碼的較小值。
- **牌池**：兩副完整撲克牌共 104 張洗成一靴；計分列即時顯示「剩餘牌」張數。
- **牌靴見底（剩 < 6 張）**：改由玩家自己選牌——剩牌背面排開，點 2 張當門柱、再點 1 張當第三張，翻開才知點數，照樣有輸贏；用完自動重洗一副新牌。
- **結算**：贏＝從彩池取走注金；輸＝注金（×倍率）賠進彩池。輸不會超過手上現有籌碼。
- **過牌**：不想博這道門，按「過」免費換一道新門，不花籌碼。
- **勝負條件**：**掃光彩池＝通關**；**籌碼歸零＝破產**。
- 起始籌碼 1000、彩池 1000、牌池 2 副（可在 `index.html` 頂部 `START_CHIPS` / `START_POT` / `DECK_COUNT` 調整）。
- 下注：**按住 −／＋ 連續增減**、**直接點數字欄位打字輸入金額**（Enter／點開即套用，自動夾到上限）、籌碼鈕（+10/+50/+100）或 ALL 一鍵梭哈；下注隨時可調。對柱時改用「猜大／猜小」鈕開牌。

## 技術

- 單一檔案 `index.html`：HTML + CSS + 原生 JavaScript，無框架、無外部資源。
- 規則引擎為純函式（`judgeNormal` / `judgePair`），載入時跑內建斷言自測（結果見瀏覽器 console）。
- 音效用 WebAudio 即時合成，無音檔；預設開啟，首次點擊後啟用。
- 字體：拉丁字（標題、數字、牌面點數）用 Hoefler Text 襯線、中文用 Noto Serif／宋體，純系統字、零外部依賴。
- 視覺：經典賭場綠氈、金色描邊、撲克質感；發牌／翻牌／勝負光暈動畫純 CSS。
